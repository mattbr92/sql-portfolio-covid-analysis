# COVID-19 Data Analysis with SQL (BigQuery)

## Project Overview
This repository contains a series of SQL queries designed to analyze the global and regional impact of the COVID-19 pandemic. Using the **Google Cloud BigQuery Public Dataset**, I performed data cleaning, mortality rate calculations, and trend identification.

## 🛠️ Skills & Technologies
* **Platform:** Google BigQuery
* **Advanced SQL Techniques:**
  * **CTEs (Common Table Expressions)** for modular code.
  * **Aggregate Functions:** `SUM`, `MAX`, `ROUND`.
  * **Data Filtering:** `WHERE`, `IN` operators.
  * **Calculated Metrics:** Mortality Rate percentages.

## Key Insights
* **Regional Hotspots:** Identified provinces in Mexico with high mortality counts using CTEs.
* **Relative Impact:** Calculated mortality rates in Colombia, showing that rural areas often face higher risks than major urban centers.
